margin, border and padding

Margin, border and padding are all part of the box model of CSS. The box model is a way of thinking about the design and layout of a page, breaking down each individual element into boxes. Any HTML element can be considered a box, and we can use these boxes in CSS to change the design and layout of our page. HTML elements include things like: title, heading, image, paragraph etc. We will be using the analogy of a framed picture on a wall to explore this concept.

The border is the edge of the element. If you think about a framed picture, the border is like the wooden frame itself. Borders by default are ‘invisible’, not having any size or width, but this can be changed with CSS. Even if the borders remain ‘invisible’, margin and padding still apply.
